ITWorks WLAN Security Checklist

Security Measure Description Completed Notes
Change Default SSID Change the default SSID to something unique and not related to the organization or location.
Change Default Admin Credentials Change the default administrator credentials for the wireless access points to strong, unique passwords.
Disable Unused Services Disable any unused services and ports on the wireless access points to reduce the attack surface.
Enable Encryption Enable WPA2 or WPA3 encryption to protect the wireless network from unauthorized access.
Limit Access Limit access to the wireless network by using MAC filtering and strong authentication methods such as EAP-TLS or PEAP.
Regularly Update Firmware Regularly update the firmware of the wireless access points to patch vulnerabilities and improve performance.
Monitor for Rogue Access Points Monitor the wireless network for rogue access points and take immediate action if any are discovered.
Regularly Review Logs Regularly review the logs of the wireless access points to detect any suspicious activity.
